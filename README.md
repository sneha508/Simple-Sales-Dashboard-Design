Sales Dashboard – Task 08

A simple and interactive sales dashboard built using Power BI, based on the Superstore dataset.

Included Files
Task8_Dashboard.pbix – Power BI source file
dashboard_screenshot.png – Dashboard preview
Sales_Dashboard_Task8.pdf – Final exported deliverable
Superstore_Sales.csv – Dataset used



Dashboard Overview
Monthly Sales Trend (Line Chart)
Sales by Region (Bar Chart)
Sales by Category (Donut Chart)
Interactive filters for Region & Category

Usage
Open the .pbix file in Power BI Desktop to view or modify the dashboard.
